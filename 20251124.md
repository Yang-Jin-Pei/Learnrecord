# 2 Github 上传
一种是在VScode本地传进Github，另一种是在Github的 `repositories` 云端进行更改。
## 2.1 VScode 本地上传
本地的 VScode 打开终端，在终端操作上传。
### 2.1.1 终端配置Git信息
```bash
# 配置用户信息（首次使用需要）
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"
```
### 2.1.2 VScode 上传
```bash
# 添加远程仓库
git remote add origin https://github.com/用户名/仓库名.git

# 推送到 GitHub
git push -u origin main
```
# 2.2 Git云端
打开 `repositories` 界面，输入`>`，即可跳转到云端编辑。
